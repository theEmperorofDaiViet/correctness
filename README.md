# Getting Started

## Prerequisites
To use this module, your system needs to have:
* numpy
  ```sh
  pip install numpy
  ```
* pandas
  ```sh
  pip install pandas
  ```

## Installation
You can install this module by cloning this repository into your current working directory:
```sh
git clone https://github.com/theEmperorofDaiViet/correctness.git
```
